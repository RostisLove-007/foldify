class ErrorViewer(wx.Frame):

Constructor: ErrorViewer(parent, crease_pattern, problem_vertices)

View In Hierarchy

Окно для визуализации паттерна складок с отмеченными проблемными вершинами.

Показывает паттерн складок и красным цветом выделяет вершины, нарушающие условия Маекавы или Кавасаки.

crease_pattern (CreasePattern): Паттерн для визуализации. problem_vertices (list): Список проблемных вершин (Point2D). panel (wx.Panel): Панель для рисования.
